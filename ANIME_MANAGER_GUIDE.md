# 动漫图片管理系统使用指南

## 📋 概述

本系统帮助你修复和管理动漫图片，通过集成Bangumi API获取真实的动漫封面和信息。

## 🚀 快速开始

### 1. 安装依赖

```bash
cd /workspace/projects
pnpm install
```

### 2. 启动后端服务器

```bash
pnpm run server
```

后端服务将在 http://localhost:3001 启动

### 3. 启动前端开发服务器

```bash
pnpm run dev
```

前端服务将在 http://localhost:5000 启动

## 🔑 登录管理后台

1. 访问前端网站
2. 使用管理员账号登录：
   - 用户名：`admin`
   - 密码：（需要先注册admin账号）

3. 登录后，在用户菜单中点击"动漫图片管理"

## 🎯 功能使用说明

### 1. 浏览需要修复的动漫

- 左侧显示所有使用picsum.photos随机图片的动漫
- 每页显示20个，支持分页浏览
- 可以通过搜索框筛选动漫

### 2. 从Bangumi搜索真实信息

**自动搜索方式：**
1. 点击左侧任意动漫
2. 系统会自动填入该动漫标题到Bangumi搜索框
3. 点击"搜索"按钮
4. 查看搜索结果，找到匹配的动漫
5. 点击"📥 导入此条"按钮

**手动搜索方式：**
1. 如果自动搜索结果不理想，可以手动修改搜索关键词
2. 尝试使用日文原名或更准确的中文译名
3. 点击搜索后查看结果

### 3. 手动编辑

如果Bangumi搜索不到结果，可以手动编辑：

1. 点击左侧动漫
2. 在右侧"手动编辑"区域修改信息
3. 可以修改：
   - 中文标题
   - 日文标题
   - 封面图片URL（粘贴真实图片链接）
   - 工作室
   - 年份
   - 评分
4. 点击"💾 保存修改"

### 4. 支持的图片来源

**推荐来源：**
- Bangumi：https://bgm.tv/
- MyAnimeList：https://myanimelist.net/
- AniList：https://anilist.co/
- 动漫官方网站

**如何获取图片URL：**
1. 访问上述网站搜索动漫
2. 找到动漫封面图片
3. 右键图片 → "复制图片地址"
4. 粘贴到"封面图片URL"输入框

## ⚙️ API接口说明

### 管理接口

#### 获取需要修复的动漫列表
```
GET /api/admin/anime/needs-fix?page=1&limit=20&search=
```

#### 搜索Bangumi
```
GET /api/admin/anime/search-bangumi?keyword=
```

#### 从Bangumi导入
```
POST /api/admin/anime/import-from-bangumi/:animeId
Body: { bangumiId: number }
```

#### 更新动漫信息
```
PUT /api/admin/anime/:id
Body: { title, title_jp, cover_image, studio, release_year, ... }
```

## 🚩 注意事项

### 版权问题
⚠️ **重要提醒：**
- 请仅用于个人学习和研究目的
- 尊重版权，不要将图片用于商业用途
- 建议使用官方公开的宣传图片

### 使用建议
1. **分批处理**：不要一次性修改太多，建议每天处理一部分
2. **质量优先**：确保找到正确的图片，不要用错图
3. **备份数据**：修改前可以先备份数据库
4. **网络问题**：Bangumi API可能有时慢，耐心等待

### 常见问题

**Q: Bangumi搜索不到结果怎么办？**
A: 尝试以下方法：
- 使用日文原名搜索
- 简化关键词，去掉"剧场版"、"特别篇"等
- 检查是否有拼写错误
- 手动从其他网站找图片

**Q: 导入后图片不显示？**
A: 检查：
- 图片URL是否正确
- 图片链接是否支持跨域访问
- 可以尝试换一个图片源

**Q: 如何批量处理？**
A: 目前系统是逐个处理的，建议：
- 先处理热门动漫
- 按工作室或年份分类处理
- 使用搜索功能快速定位
